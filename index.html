Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://pewkodi.github.io/Leviticus/repo/ (Give it the name "Chibo Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "ChiBo addons", and install repository.chibo-1.0.2.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Chibo Repo"
  • Go into the "Repo add-ons" section in the repo

Enjoy!
